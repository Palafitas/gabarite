+++ Sobre +++
Projeto responsável por requisitar api do site gabarite e obter a questão correta através do id da questão e uma alternativa qualquer.
Site: https://www.gabarite.com.br  

+++ Uso +++
chmod +x gabarite.sh
./gabarite.sh {ID_QUESTAO} {ALTERNATIVA_QUALQUER}

- {ID_QUESTAO} é o código ID da questão, normalmente no começo da questão, e.g: 208935 é o id dessa https://www.gabarite.com.br/questoes-de-concursos/208935-questao);
- {ALTERNATIVA_QUALQUER} é uma alternativa qualquer em minúsculo, e.g: d
